## 一、卦象能量定性（Volatility Filter）

## 二、世财博弈辩证法（Subject vs. Profit）

### 世爻画像

世爻居三爻，兄弟甲辰（土），临白虎。

- 月建寅木克辰土 → 世爻受月克，基础面承压
- 起卦日午火生辰土 → 日建生扶，短期有撑
- 白虎主肃杀、下跌 → 世爻气质偏空

### 用神（妻财）画像

妻财双现：五爻癸亥（水）、初爻甲子（水）。

- **初爻甲子：旬空！** → 底层财力虚空，根基不牢
- **五爻癸亥**：不空，但寅月木泄水气 → 财力被耗
- 妻财属水，世爻属土 → **世克财格局**

### 逻辑检查（Logic Check）

世爻（土）克妻财（水），这既非"生世克财"也非"克世生财"，而是**世爻直接压制财爻**——

- 关注度/成交量（世）在耗损利润（财）
- 市场有人气但赚不到钱
- 越热闹越亏钱的格局

**通关检查**：子孙酉金（上爻，应位）可作通关——土生金生水。但子孙在应位（外部力量），且不发动，通关力量**被动、有限**。只有日辰引动子孙时，通关才会短暂开启。

### 世财博弈定性

> **结论：世克财 + 财爻一空一弱 + 兄弟双动劫财 = 财爻处于被围攻状态，整体偏空。**

## 五、三维推演矩阵（3-Scenario Matrix）

### 场景 A：假设股价处于高位/超买区（Risk On）

**核心诊断：兄弟劫财 + 泰化丰 = 经典出货卦**

- 四爻兄弟发动化午火回头生 → 兄弟得力后劫财：**主力借利好出货**
- 二爻官鬼化兄弟 → 政策面/消息面出现波动，资金趁机撤离
- 泰化丰 = 行情已经"丰满"，到达了"日中"位置，接下来是"则昃"（回落）
- 世克财格局在高位 → 成交量放大（世旺）但股价不涨反跌（财受克）→ **天量见天价，放量滞涨**
- 妻财空亡 → 账面利润是虚的，不兑现就会蒸发

**高位剧本**：这2.5天是派发窗口，第一个半日（02/16）或许还维持表面繁荣（子孙生财），但02/20和02/23连续两日兄弟值日，筹码加速流出，可能出现单日2-3%级别的回调。

**策略**：⛔ **逢高坚决减仓，不可恋战。02/16半日若有反弹即是最佳离场窗口。**

### 场景 B：假设股价处于低位/超跌区（Value On）

**核心诊断：官鬼旺而化兄弟 = 利空最后一波洗盘，但财弱难立即反转**

- 官鬼寅木得月建极旺 → 在低位意味着利空已充分释放，恐惧情绪已达极值
- 官鬼化兄弟 → 利空消息转化为最后的抛压（弱手割肉），筹码从散户流向主力
- 兄弟空动 → 洗盘力度"雷声大雨点小"，跌幅有限
- 伏神父母乙巳（禄神）→ 暗中有政策利好尚未释放，底部有支撑
- **但**：妻财空亡且弱 → 即便是底部，也不会立刻反弹，而是磨底

**低位剧本**：02/16半日可能小幅企稳（子孙生财），02/20可能再探一次底（兄弟值日、丑克亥），02/23继续在底部震荡（辰自刑世爻，纠结）。整体是**最后洗盘→磨底**的过程，不是V型反转。

**策略**：✅ **可分批左侧布局，但不要一次性重仓。02/20若出现恐慌性下跌反而是较好的吸筹机会。耐心等待节后通关子孙被激活（出空/合开）后的真正反弹。**

---

### 场景 C：假设股价处于中枢/震荡区（Breakout Watch）

**核心诊断：两爻齐动打破震荡，但方向偏向下破**

- 双动爻带来的能量足以打破此前的横盘格局
- 但两个动爻的化爻方向（均指向兄弟增强）明确偏空
- 世克财 + 兄弟劫财 → 如果要选方向，大概率**向下突破**
- 泰化丰：泰为平衡，丰为极端——平衡被打破后走向极端
- 唯一变数是02/16子孙酉金值日，若当日放量上攻，可能形成诱多后再下杀的走势

**中枢剧本**：02/16半日小幅冲高（或平稳），给出假突破信号。02/20兄弟值日，向下打破中枢，确认方向。02/23延续下行或在新的低位震荡。

**策略**：⚠️ **如持有仓位，02/16当天设好止损/止盈。如空仓观望，不要因02/16的平稳而追入——等02/20方向确认后再做决策。**
